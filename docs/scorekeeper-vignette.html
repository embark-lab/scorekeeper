A Scorekeeper Vignette

The following example depicts how to use the scoresheet package for manipulating data and saving some scored data. This example will use 6 items used to assess binge eating behavior at age 14 in the Avon Longitudinal Study of Parents and Children ALSPAC. You can use load this synthetic data and scoresheet in the package to test the example and familiarize yourself with the package.

First, we load the scorekeeper package:

library(scorekeeper)
